Thesis

My undergraduate thesis, “Performance Analysis of Generalized and Specified News Category SVM Models for Fake News Detection Using Categorized Datasets and Sentiment Analysis”, focuses on developing a fake news detection website designed to identify fabricated content. The system features a web interface built with Gradio and leverages Support Vector Machine (SVM) models implemented in Python with Scikit-learn. To improve accuracy and reliability, I integrated sentiment analysis, web scraping, TF-IDF, and confidence scoring into the detection pipeline. The study also highlights a comparative analysis between generalized and category-specific models, providing insights into their performance in classifying news more effectively.
